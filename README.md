# Carboni-MES
>**조원: 나현주(팀장), 조준희(부팀장), 김영민, 김주환, 유해정**<br>
>제품 생산의 전 과정을 최적화하는 통합생산관리시스템(MES)을 통하여 제품의 주문부터 출하단계까지 모든 내용을 데이터화 할 수 있고, 실시간 모니터링을 통해 생산 진행상황을 확인할 수 있다.
>2022.06.20.~2022.08.10
